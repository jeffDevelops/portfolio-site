Commitment
Issues

Github Companion

React, Firebase

Commitment Issues helps new developers build healthy version control habits by reminding them to push to remote often. Time is money, so don't waste it by reverting back to a version last backed up hours, days ago, or maybe not at all! Simply log in via the GitHub OAuth provider and you're ready to track the repository you want. Select an interval (up to four times an hour, or up to 8 times per workday), and hack away... Commitment Issues will nudge you when it's time to commit and push!

Binaural

Browser Game

HTML, CSS, Vanilla JS DOM API

Binaural teaches aural skills to music theory students. It puts 60 seconds on the clock for two players to guess as many aural intervals as possible. Whoever makes the most correct guesses wins!

Same

Full Stack Web Application

MongoDB, Express, Node

Same is an application built with Node, MongoDB, and Express, designed to make networking easier. Create a digital representation of your event by importing via the Meetup API. When your guests arrive, they take a photo of themselves, add a short description, and notify people with common interests that they want to chat.

Color

Full-Stack Single Page Application

SQL, Express, Angular, Node

Color removes the subjectivity from picking branding colors — or at least, some of it. It uses a hand-coded RESTful API to access colors mapped psychologically by the Dewey Color System. Select from our list of attributes that match your brand's identity, and get a base color for your branding pallette. Or, select one of your favorite colors and see if its psychological associations match your company's culture.
